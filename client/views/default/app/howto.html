How To
Purpose of the App
The purpose of this sample app is to demonstrate how to:
•
View issue list, view details of issues
•
Add new Issues, Update existing issues
Prerequisites
•
SmartUX Studio installed and setup.
Steps to run the app
1.
Open command prompt/terminal
2.
Go to directory
dataStore/templates/
MantisClassic/server
3.
Check node_modules subdirectory exists,
If not follow Steps a-c, otherwise continue to Step 4
a.
Run the command npm install
b.
Verify node_modules directory is created and present
c.
This will ensure node modules dependencies are installed
4.
Start the server application by typing in terminal: npm start.
5.
Click the Preview button to preview the app in the emulator.
6.
Switch/toggle off Mock Server Data to view live-data.
7.
You will see the list of issues. Click on name or ID of any issue to see the details of that issue
8.
Click on the Plus icon at the top right corner, to add a new issue
9.
Fill in all the required details and click on the Submit Report button to submit the issue
10.
Click on the Edit icon given against issue, to update that issue
11.
Modify the issue data and click on Update Information button. It will update the issue and a message will be shown if the issue is successfully updated
